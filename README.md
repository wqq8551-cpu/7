# Arix Signature Christmas Experience

A high-fidelity, interactive 3D holiday experience built with React Three Fiber.

## Features
- **Cinematic 3D Rendering**: Custom lighting, bloom, and vignette effects.
- **Interactive Assembly**: Watch the tree assemble from chaos to structure.
- **Magical State**: Ignite the tree with particle effects and golden glow.
- **Responsive Design**: Works on desktop and mobile.

## How to Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## How to Deploy

To publish this project to a shareable link (GitHub Pages):

1. Make sure you have created a repository on GitHub.
2. Push your code to the `main` branch.
3. Run the deploy script:
   ```bash
   npm run deploy
   ```
